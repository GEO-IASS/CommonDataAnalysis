double CDA::EM< data_T, theta_T >::EMstep (  )  [inline, protected]

One iteration.

DESCRIPTION

This is the basic EM (not extended EM etc.) algorithm, by virtue of calls to virtual functions you can adapt the procedure to your problem.

SIDE-EFFECTS

This will update the model parameters stored in the EMThetas class.

Rückgabe:
The new log-likelihood value after performing the iteration.
